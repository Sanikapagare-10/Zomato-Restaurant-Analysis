# 🍽️ Zomato Restaurant Analysis and Rating Prediction

## 📌 Project Overview

This project analyzes the Zomato restaurant dataset to uncover business insights and predict restaurant ratings using Machine Learning.

The project covers the complete Data Analytics workflow, including Data Cleaning, Exploratory Data Analysis (EDA), SQL Analysis, Data Visualization, and Machine Learning.

---

## 🎯 Objectives

- Analyze restaurant data to identify business trends.
- Perform SQL-based business analysis.
- Visualize restaurant patterns using Python.
- Build a Machine Learning model to predict restaurant ratings.
- Generate business recommendations based on insights.

- Customer votes significantly influence restaurant ratings.
- Restaurant location impacts popularity.
- Premium restaurants generally achieve higher ratings.
- Cuisine diversity attracts a wider customer base.
